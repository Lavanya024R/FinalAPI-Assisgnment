# Medical Appointment System

A simple **FastAPI-based Medical Appointment System** to manage doctors, schedule appointments, and track consultations.

## Features

- View all doctors
- Get doctor details by ID
- Doctor summary data
- Filter, search, sort, and paginate doctors
- Add, update, and delete doctors
- Create appointments
- Confirm, cancel, and complete appointments
- View active appointments
- View appointments by doctor
- Search, sort, and paginate appointments
- Combined doctor browse endpoint with filter + sort + pagination

## Tech Stack

- Python
- FastAPI
- Pydantic
- Uvicorn
